# Xiaohongshu Playbook

## Load when

Use this reference for Xiaohongshu notes, creator briefs, Pugongying, Juguang Feed/Search, search interception, keyword matrices, and station-to-Douyin/Tmall landing.

## Xiaohongshu role

Xiaohongshu should build content trust, search decision, scenario mindshare, category education, creator proof, brand-word growth, category-word occupation, competitor-word interception, and off-platform landing to Tmall or Douyin.

## Note matrix

Use a portfolio, not one generic note type:

- Pain-solution.
- Real review.
- Ingredient/material explanation.
- Before-after comparison where compliant.
- Competitor comparison where compliant.
- Avoidance/education.
- Good-items collection.
- Scenario checklist.
- Audience-specific guide.
- Repurchase recommendation.
- Gift/campaign recommendation.

Avoid turning every note into hard selling. Match content depth to user decision difficulty.

## Keyword matrix

Build keyword groups:

- Brand words.
- Category words.
- Benefit/function words.
- Scenario words.
- Problem words.
- Audience words.
- Competitor words.
- Price words.
- Ingredient/material words.
- Season words.
- Gift words.

Decide which words belong in note titles, body, comments, search ads, creator briefs, Tmall titles, and Douyin product-card/search landing.

## Feed + Search split

Feed handles discovery, seeding, break-out content, creator amplification, audience expansion, and scenario mindshare.

Search handles brand-word defense, category occupation, competitor/problem interception, high-intent capture, and in-platform decision landing.

If Feed gets interaction but search does not rise, inspect keyword coverage, note title clarity, creator relevance, comment intent, and landing path.

If Search captures clicks but does not convert, inspect price consistency, proof, reviews, off-platform landing, and SKU match.

## Creator brief output

For creator plans, specify:

- Target audience and use scenario.
- Product role and cannot-say claims.
- Core proof points and acceptable evidence.
- Required keywords.
- Forbidden exaggeration and compliance risks.
- Visual scene and opening hook.
- Comment guidance.
- Landing channel and search words to defend.
- Measurement window and review metrics.

## Review metrics

Track exposure, click, interaction, save, comment quality, search lift, brand-word/category-word volume, product-detail or store visits if available, off-platform landing signal, content reuse value, and audience asset value.
